Return Policy and Cancelation Policy - Rules.

USER

User can claim about refund when event or live streaming not starting from organizer side.

Here some criteria about refund,

  1. User claim before 30 days of booking return 90% amount.
  2. User claim between 21-30 days of booking return 70% amount.
  3. User claim between 11-20 days of booking return 30% amount.
  4. User claim between 6-10 days of booking return 10% amount.
  5. User claim between 1-5 days of booking no return any amount.

If Event reschedule then open two options for user. One option is cancelled reschedule event and ask for refund or 2nd is continue with reschedule event.

If user cancel reschedules event ask for refund, then total amount will be refund. And 6% minus amount on event organizer it will be collect from deposit or F-coin.

When event reschedule generate new ticket barcode.

EVENT ORGANIZER

If Event cancel then refund total amount to users and 6% amount will be minus on event organizer. If organizer have enough coin or deposit then deduct 1st deposit then coin. And show on organizer dashboard.

Every registering events reserve 10% of total amount before start events. This condition will apply when Event organizer cancel event or reschedule. At first time this situation was happened deposited flow work every time for particular organizer. This situation totally depends on first booking otherwise these terms not Appling.

When one booking Is done and If Event cancel then the deposit amount not refundable to organizer. When no one is booking of event then Deduct 6% charge from deposit or F-coin then release.

If Event not cancel then the deposit amount returns back to organizer after deduction of 5% bank charge (as per razor pay payment gateway).

If Event reschedule then open two options for user. One option is cancelled reschedule event and ask for refund or 2nd is continue with reschedule event.

If user cancel reschedules event ask for refund, then total amount will be refund. And 6% minus amount on event organizer it will be collect from deposit or F-coin.

LIVE STREAMING ORGANIZER

If paid Live streaming cancel from organizer then refund total amount to users and not refund to organizer any amount that was paid on live streaming creation time.

If free Live streaming cancel from organizer, then not refund to organizer any amount that was paid on live streaming creation time.

If paid live streaming reschedule then open two options for user. One option is cancelled reschedule event and ask for refund or 2nd is continue with reschedule event.

If free live streaming reschedules then open two options for user. One option is cancelled reschedule event or 2nd is continue with reschedule event.

Note: This Calculation show to admin and admin reports and also printable.
